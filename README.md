# WinSpace

WinSpace 是一款基于 WPF 和 C# 开发的 Windows 桌面增强工具。它的核心目标是将 macOS 标志性的窗口和工作区管理体验（如 Mission Control、Spaces）原汁原味地带到 Windows 系统中，帮助用户大幅提升多任务处理的效率。

### 核心概念

- **多空间 (Spaces)**：你可以把它理解为多个“虚拟桌面”。你可以将不同类型的工作分配到不同的空间中（例如：空间 1 放浏览器查找资料，空间 2 放代码编辑器写代码，空间 3 放微信/QQ等聊天软件）。通过左右切换空间，你可以将不同的任务物理隔离，保持专注，避免在杂乱无章的重叠窗口中迷失。
- **控制中心 (Mission Control)**：这是一个全局的鸟瞰视图。当你在多个空间和几十个窗口中工作，突然找不到某个应用时，只需一个快捷键，就能在屏幕上平铺展示所有的工作区以及当前空间内的所有窗口缩略图。它让你对全局状态一目了然，并能一键跳转到你想要的任何窗口或工作区。

### 默认快捷键

WinSpace 提供了几组默认的快捷键，帮助你流畅地在不同任务间穿梭（所有快捷键都可以在设置中自定义）：

- `Ctrl` + `Alt` + `F`：**切换/恢复当前窗口到工作空间**。不仅是最大化，更是让你瞬间沉浸在当前任务中。
- `Ctrl` + `Alt` + `←` (左方向键)：**向左切换空间**。切换到左侧的工作空间。
- `Ctrl` + `Alt` + `→` (右方向键)：**向右切换空间**。切换到右侧的工作空间。
- `Ctrl` + `Alt` + `↑` (上方向键)：**进入控制中心**。当窗口太多找不到时，按下它，一切尽收眼底，鼠标点击即可快速跳转。
